Pergunta 1: 
A diferença entre git add. e git commit -m "mensagem" é:
O git add. adiciona os arquivos modificados pra uma "área de preparação", onde estarão os arquivos que serão salvos no próximo commit  
O git commit cria um registro com uma mensagem do que foi alterado com o git add
